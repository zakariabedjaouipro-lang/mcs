## 🔧 كيفية التعديل والتطوير

### إضافة بيانات جديدة

إذا أردت إضافة عمود جديد في الجدول البيانات، اتبع هذه الخطوات:

#### 1. أضف Event جديد في `admin_event.dart`
```dart
/// Load new data
class LoadNewData extends AdminEvent {
  const LoadNewData();
}
```

#### 2. أضف State جديد في `admin_state.dart`
```dart
/// New data loaded
class NewDataLoaded extends AdminState {
  const NewDataLoaded(this.data);
  final List<dynamic> data;

  @override
  List<Object?> get props => [data];
}
```

#### 3. أضف Handler في `admin_bloc.dart`
```dart
Future<void> _onLoadNewData(
  LoadNewData event,
  Emitter<AdminState> emit,
) async {
  emit(const AdminLoading());
  try {
    final data = await _supabaseService.fetchAll('table_name');
    emit(NewDataLoaded(data));
  } catch (e) {
    emit(AdminError('فشل التحميل: $e'));
  }
}
```

#### 4. أضف تبويب جديد في `super_admin_dashboard_v2.dart`
```dart
Tab(text: 'البيانات الجديدة'),

// ثم أضف عمود جديد في TabBarView:
_buildNewDataTab(context, state, isArabic),

// وأخيراً أنشئ الدالة:

  BuildContext context,
  AdminState state,
  bool isArabic,
) {

    // أظهر البيانات هنا
  }
  return const SizedBox.shrink();
}
```

### ما يجب اختباره

- ✅ تحميل البيانات عند فتح الشاشة

- ✅ عرض رسالة الخطأ عند فشل التحميل
- ✅ زر "إعادة محاولة" يعمل
- ✅ التنقل بين التبويبات بسلاسة

## 🐛 حل المشاكل الشائعة

| المشكلة | الحل |
|--------|------|
| **البيانات لا تحمل** | تحقق من اتصال قاعدة البيانات |
| **رسالة خطأ عند الفتح** | تأكد من وجود جداول مناسبة في Supabase |
| **الألوان لا تظهر صحيحة** | تحقق من إعدادات الموضوع (Theme) |
| **الصفحة بطيئة** | قد تحتاج لترقية الأداء (pagination) |
| **البيانات قديمة** | أضف زر "تحديث" يدوي |

## 💡 نصائح للأداء الأفضل

1. **استخدم caching** - احفظ البيانات محلياً
2. **lazy loading** - حمّل البيانات عند الحاجة فقط
3. **pagination** - لا تحمّل كل البيانات دفعة واحدة
4. **debouncing** - لا تحدّث الواجهة بسرعة كبيرة
