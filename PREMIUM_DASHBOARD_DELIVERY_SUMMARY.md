## 🏗️ Architecture

### **Layer Structure**
```
Presentation Layer
├── PremiumSuperAdminDashboard (Widget)
├── _buildSidebar() → Sidebar UI
├── _buildTopAppBar() → App Bar UI
├── _buildMainContent() → Content Area
└── Helper methods (Icons, Colors, Animations)

State Management Layer
├── ThemeBloc (for theme switching)
├── LocalizationBloc (for language switching)
└── ApprovalBloc (for user approvals)

Routing Layer
├── AppRouter configuration
├── GoRouter routes
└── Navigation guards

Data Layer
├── RemoteDataSources
├── Repositories
└── Models
```

### **Design Pattern**
- ✅ Clean Architecture (Data → Domain → Presentation)
- ✅ BLoC pattern for state management
- ✅ Dependency Injection (GetIt)
- ✅ Repository pattern
- ✅ Model classes with JSON serialization
